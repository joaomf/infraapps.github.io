Privacy Policy for GPS Photo Report

GPS Photo Report is designed to help users create professional photo reports with GPS location, watermark and PDF export. This privacy policy explains what the app accesses, how it is used, and the current privacy approach for the app.

Política de Privacidade

Política de Privacidade do GPS Photo Report

O GPS Photo Report foi desenvolvido para ajudar usuários a criar relatórios fotográficos profissionais com localização GPS, marca d’água e exportação em PDF. Esta política explica quais recursos o app acessa, como eles são usados e qual é a abordagem atual de privacidade do aplicativo.

1. Information the app uses

GPS Photo Report may access the following device features and user content when needed for the app to function:

  • Camera, to capture photos inside the app.
  • Photo Library, to save photos to the app album and display existing photos or videos selected by the user.
  • Location, when the user wants to add GPS data to photos or videos, or when a geolocated photo is taken in the app.

1. Informações que o app utiliza

O GPS Photo Report pode acessar os seguintes recursos do dispositivo e conteúdos do usuário quando necessário para o funcionamento do app:

  • Câmera, para capturar fotos dentro do aplicativo.
  • Biblioteca de Fotos, para salvar fotos no álbum do app e exibir fotos ou vídeos existentes selecionados pelo usuário.
  • Localização, quando o usuário quiser adicionar dados de GPS a fotos ou vídeos, ou quando uma foto geolocalizada for capturada no app.

2. How the app uses that information

The app uses the accessed data only to support the features described on the product page and in the App Store listing, including:

  • Adding GPS location to existing photos.
  • Adding GPS location to existing videos.
  • Taking geolocated photos with watermark.
  • Saving photos to the app album.
  • Allowing the user to select photos and generate PDF reports.
  • Sharing selected photos and generated PDF reports.

2. Como o app usa essas informações

O app utiliza os dados acessados apenas para suportar os recursos descritos na página do produto e na listagem da App Store, incluindo:

  • Adicionar localização GPS a fotos existentes.
  • Adicionar localização GPS a vídeos existentes.
  • Tirar fotos geolocalizadas com marca d’água.
  • Salvar fotos no álbum do app.
  • Permitir que o usuário selecione fotos e gere relatórios em PDF.
  • Compartilhar fotos selecionadas e relatórios em PDF gerados.

3. Data sharing

GPS Photo Report is not described here as a cloud-processing or account-based platform. The workflow presented for the current version is based on direct use on the user's device.

When the user chooses to share content, the app shares only the items explicitly selected by the user within the app workflow, such as selected photos or generated PDF reports.

3. Compartilhamento de dados

O GPS Photo Report não é descrito aqui como uma plataforma baseada em conta ou em processamento na nuvem. O fluxo apresentado para a versão atual é baseado no uso direto no dispositivo do usuário.

Quando o usuário escolhe compartilhar conteúdo, o app compartilha apenas os itens explicitamente selecionados dentro do fluxo do aplicativo, como fotos escolhidas ou relatórios em PDF gerados.

4. Data collection

Based on the current product description prepared for release, this policy does not claim broad personal data collection, tracking, analytics profiling, or external sale of user data.

If those behaviors are added in future versions, this privacy policy should be updated before release.

4. Coleta de dados

Com base na descrição atual do produto preparada para lançamento, esta política não afirma coleta ampla de dados pessoais, rastreamento, perfilamento analítico ou venda externa de dados do usuário.

Se esses comportamentos forem adicionados em versões futuras, esta política de privacidade deverá ser atualizada antes da publicação.

5. User control

Users can control app permissions through their device settings, including camera, photos and location permissions. If a permission is denied, some app features may not work as expected.

5. Controle do usuário

Os usuários podem controlar as permissões do aplicativo nas configurações do dispositivo, incluindo câmera, fotos e localização. Se uma permissão for negada, alguns recursos do app podem não funcionar como esperado.

6. Children’s privacy

GPS Photo Report is presented as a professional utility app for technical documentation, inspection and field workflows. It is not specifically directed to children.

6. Privacidade de crianças

O GPS Photo Report é apresentado como um app utilitário profissional para documentação técnica, inspeções e fluxos de trabalho em campo. Ele não é direcionado especificamente a crianças.

7. Changes to this policy

This policy may be updated to reflect future changes in app features, sharing behavior, account structure, analytics, cloud usage, or legal requirements.

7. Alterações nesta política

Esta política pode ser atualizada para refletir mudanças futuras nos recursos do app, no comportamento de compartilhamento, na estrutura de contas, em análises, uso de nuvem ou exigências legais.
